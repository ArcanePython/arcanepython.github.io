+   DESCRIPTION   +                                              
+=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-+          
Custom skybox with underwater scene. First I generated images in 3ds max then used those images with an added terrain in JKA to make the skybox.
Size of textures: 1024x1024 pixels.
Shader file included.


+   INSTALLATION INSTRUCTIONS   +                                
+=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-+          

Unzip the uw_sky.zip archive.
Put the underwater folder to your Gamedata/base/textures folder.                             
Put the uw_sky.shader file to your Gamedata/base/shaders folder.

Don't forget to add the shader to your shaderlist.txt in the same folder.
In Radiant you can find the shader as uw_sky in the skies folder.
